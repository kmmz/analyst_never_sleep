# Effy's Investment Analyst Plugin Marketplace

A curated collection of investment research plugins for [Claude Cowork](https://claude.ai).

## Plugins

### investment-analyst

A full investment research toolkit with 5 skills:

- **Company Deep Dive** — comprehensive research on any public company or ticker
- **Daily Market Briefing** — structured morning briefing with macro & equity updates
- **Earnings Prep** — pre-earnings frameworks and post-earnings analysis
- **Investment Memo** — write up your thesis as a formal one-pager
- **Watchlist Manager** — track tickers, levels, and theses in natural language

---

## How to Install (for friends)

### Step 1 — Add this marketplace

In Claude Cowork or Claude Code, run:

```
/plugin marketplace add https://github.com/kmmz/analyst_never_sleep
```

### Step 2 — Install the plugin

```
/plugin install investment-analyst@effy-plugins
```

That's it! You'll now have all 5 skills available in your Cowork sessions.

### Getting updates

When new versions are published, refresh with:

```
/plugin marketplace update effy-plugins
```

---

## Requirements

- Claude Cowork (desktop app) or Claude Code CLI
- No additional API keys required for the base plugin
- Optional: connect a Bigdata MCP or similar for live market data
